# Todo List Manager 📝

A lightweight and simple **Todo List Manager** built entirely in **Bash**. This CLI-based application allows you to manage your tasks efficiently without the need for heavy software. Perfect for users who prefer a terminal-based workflow.  

## ✨ Features

- ➕ **Add Tasks**  
  Quickly add tasks to your todo list.  

- ✅ **Mark Tasks as Done**  
  Easily mark tasks as completed.  

- ❌ **Delete Tasks**  
  Remove tasks from your list when they’re no longer needed.  

- 📜 **View Tasks**  
  Display all tasks with their completion status.  

- 🔄 **Persistent Storage**  
  Tasks are stored in a plain text file for simplicity and portability.  

## 🚀 Tech Stack

- **Language**: Bash (Bourne Again Shell)  
- **Storage**: Plaintext file (`todo.txt`)
